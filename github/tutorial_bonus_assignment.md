# Raise an issue
1. Find the issue.
2. Click `New issue`
![](./aset/01_create_issue.svg)
3. Create new issue and click `Create`

# Volunteer to fix an issue
4. State your intention to fix the issue, so that other members do not target the same issue. Click `Comment`

# Fix the issue
5. Click `Code` to start editing. If you cannot click `Code`, that means you are not yet a collaborator in this repo.
